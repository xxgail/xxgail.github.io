HTML

  1. html 标签不分大小写，但建议小写。

  2. 确定逻辑部分:如网页中的独立的栏目板块。相当于一个容器。
    <div id = "版块名称">...........</div>

  3. 创建表格的四个元素:(没有表格线)
    table、tbody、tr、th、td
    3.1、<table>…</table>：开始和结束。
    3.2、<tbody>…</tbody>：当表格内容非常多时，表格会下载一点显示一点，但如果加上标签后，这个表格就要等表格内容全部下载完才会显示。如右侧代码编辑器中的代码。
    3.3、<tr>…</tr>：表格的一行，所以有几对tr 表格就有几行。
    3.4、<td>…</td>：表格的一个单元格，一行中包含几对…，说明一行中就有几列。
    3.5、<th>…</th>：表格的头部的一个单元格，表格表头。表头的字体默认为粗体居中。
    3.6、表格中列的个数，取决于一行中数据单元格的个数。

CSS
用于定义html内容在浏览器内的显示样式

  1. CSS样式有选择符和声明组成，声明由属性和值组成，属性与属性之间用；隔开。

  2. CSS的三种样式。优先级:内联式>嵌入式>外部式
    2.1 内联式:<p style = "color : red">这里文字是红色</p>
    2.2 嵌入式: 样式必须写在<style></style>之间，且一般情况下写在<head></head>之间。
    2.3 外部式:<link href=**"base.css"** rel="stylesheet" type="text/css" />。把CSS代码写在一个单独的外部文件中，以”.css”为扩展名。<link>标签在<head>标签之内。

  5. 类选择器和id选择器的区别: id选择器只能在文档中使用一次。类选择器可以设置多个样式。

  6. 子选择器(>)，用于选择指定标签元素的第一代子元素。.food>li{color:red}

  11. CSS中继承的意思:它允许样式不仅应用于某个特定html标签元素，而且应用于其后代，但只限定某些样式。

  12. 如果标签匹配了两个或以上的样式代码，用权值高的CSS样式。
    权值的规则为:标签的权值为1，类选择符的权值为10，ID选择符的权值最高为100。
    权值一样时处在最后面的css样式会被应用。
    前面的css样式优先级为:内联样式表（标签内部）> 嵌入样式表（当前文件中）> 外部样式表（外部文件中）。

  13. 可以在样式代码中分号的前面加上!important来设置最高权值。

  15. 元素分类:
    常用的块状元素有：
    <div>、<p>、<h1>...<h6>、<ol>、<ul>、<dl>、<table>、<address>、<blockquote> 、<form>
    常用的内联元素有：
    <a>、<span>、<br>、<i>、<em>、<strong>、<label>、<q>、<var>、<cite>、<code>
    常用的内联块状元素有：
    <img>、<input>

  16. 块级元素特点：
    将内联元素a转换成内联元素:a{display:block;}
    1、每个块级元素都从新的一行开始，并且其后的元素也另起一行。（真霸道，一个块级元素独占一行）
    2、元素的高度、宽度、行高以及顶和底边距都可设置。
    3、元素宽度在不设置的情况下，是它本身父容器的100%（和父元素的宽度一致），除非设定一个宽度。

  17. 内联元素特点：
    将块状元素转换成内联元素:div{display:inline;}
    1、和其他元素都在一行上；
    2、元素的高度、宽度及顶部和底部边距不可设置；
    3、元素的宽度就是它包含的文字或图片的宽度，不可改变。

  18. 内联块状元素特点：
    转化成内联块状元素:display:inline-block
    1、和其他元素都在一行上；
    2、元素的高度、宽度、行高以及顶和底边距都可设置。

  19. <div>、<ul>、<ol>、<p>、<h>、<table>等等这些！块级标签！都具备盒子模型的特征。
    盒子与内容之间的距离（内边距，内填充）：padding
    盒子与另一个盒子之间的距离（外边距）：margin
    盒子的边框宽度：border
    盒子模型的内填充、外边距和边框都有4个方向，上（top），右（right），下（bottom），左（left），盒子模型实际高度=内容高度+上下内填充高度+上下边框高度，宽度同理。

  20. 盒子模型的边框是围绕着内容以及补白的线，有粗细，样式和颜色(边框)三个属性。

  23. 元素的三种布局模型:
    1、流动模型（Flow）
    2、浮动模型 (Float)
    3、层模型（Layer）
  24. 流动模型:流动（Flow）是默认的网页布局模式。也就是说网页在默认状态下的 HTML 网页元素都是根据流动模型来分布网页内容的。

  25. 流动布局模型具有2个比较典型的特征：
    第一点，块状元素都会在所处的包含元素内自上而下按顺序垂直延伸分布，因为在默认状态下，块状元素的宽度都为100%。实际上，块状元素都会以行的形式占据位置。如右侧代码编辑器中三个块状元素标签(div，h1，p)宽度显示为100%。
    第二点，在流动模型下，内联元素都会在所处的包含元素内从左到右水平分布显示。（内联元素可不像块状元素这么霸道独占一行）。

  27. 层模型(是html元素在网页中精确定位)有三种形式:
    1、绝对定位(position: absolute)
    2、相对定位(position: relative)
    3、固定定位(position: fixed)

  28. 绝对定位 : 将元素从文档流中拖出来，然后使用left、right、top、bottom属性相对于其最接近的一个具有定位属性的父包含块进行绝对定位。如果不存在这样的包含块，则相对于body元素，即相对于浏览器窗口。

  29. 相对定位 : 为元素设置层模型中的相对定位，它通过left、right、top、bottom属性确定元素在正常文档流中的偏移位置。相对定位完成的过程是首先按static(float)方式生成一个元素(并且元素像层一样浮动了起来)，然后相对于以前的位置移动，移动的方向和幅度由left、right、top、bottom属性确定，偏移前的位置保留不动。

  30. 固定定位:与absolute定位类型类似，但它的相对移动的坐标是视图（屏幕内的网页窗口）本身。由于视图本身是固定的，它不会随浏览器窗口的滚动条滚动而变化，除非你在屏幕中移动浏览器窗口的屏幕位置，或改变浏览器窗口的显示大小，因此固定定位的元素会始终位于浏览器窗口内视图的某个位置，不会受文档流动影响，这与background-attachment:fixed;属性功能相同。position:fixed

  31. Relative与Absolute组合使用:相对于其他元素进行定位。

  32. 盒模型代码简写:
    1、如果top、right、bottom、left的值相同 可缩写为：
    margin:10px;
    2、如果top和bottom值相同、left和 right的值相同，可缩写为：
    margin:10px 20px;
    3、如果left和right的值相同 可缩写为：
    margin:10px 20px 30px;
    注意：padding、border的缩写方法和margin是一致的。
    4、设置的颜色是16进制的色彩值时，如果每两位的值相同，可以缩写一半。如:p{color:#336699}=p{color:#369}。
    5、字体在缩写时 font-size 与 line-height 中间要加入“/”斜扛。

  35. 水平居中设置-行内元素:

  1. 水平居中设置-定宽块状元素:可以通过设置“左右margin”值为“auto”来实现居中。

  2. 水平居中设置-不定宽块状元素方法:
    1.加入 table 标签，利用table标签的长度自适应性—即不定义其长度也不默认父元素body的长度（table其长度根据其内文本长度决定）。
    第一步：为需要设置的居中的元素外面加入一个 table 标签 ( 包括 、、 )。
    第二步：为这个 table 设置“左右 margin 居中”（这个和定宽块状元素的方法一样）。
    2.设置 display: inline 方法：与第一种类似，显示类型设为 行内元素，进行不定宽元素的属性设置，使用 text-align:center 来实现居中效果。
    3.设置 position:relative 和 left:50%：利用 相对定位 的方式，将元素向左偏移 50% ，即达到居中的目的。通过给父元素设置 float，然后给父元素设置 position:relative和 left:50%，子元素设置 position:relative 和 left: -50% 来实现水平居中。

  3. 垂直居中-父元素高度确定的单行文本:通过设置父元素的 height 和 line-height 高度一致来实现的。(height: 该元素的高度，line-height: 顾名思义，行高（行间距），指在文本中，行与行之间的 基线间的距离 )。(即两者的值一样)

  4. 垂直居中-父元素高度确定的多行文本:使用插入 table (包括tbody、tr、td)标签，同时设置 vertical-align：middle(因为td标签默认设置了该属性，所以不需要显式设置)。
  5. 为元素设置以下两个语句时会将元素变成块状元素:(元素类型为display:none的除外)
    1.position:absolute
    2.float:left或float:right
